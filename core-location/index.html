The Triforce

The three technologies that drive Core Location

The Triforce of Efficiency


Core Location gets your whereabouts using cell-tower triangulation

  • Uses cell-tower triangulation to find you
  • Most battery efficient
  • Lowest latency
  • Least accurate

The Triforce of Accuracy


Core Location gets your whereabouts using GPS satellites

  • Turns on a dedicated GPS hardware module to find you
  • Most accurate
  • Takes some time
  • Gobbles battery
  • Succeptible to tunnels

The Triforce of Power


Core Location levels up when you enable WiFi

  • Snoops nearby Wifi SSIDs to somehow figure out where you are
  • Impossible to use on its own
  • Augments Cell & GPS
  • Probably ineffective in the countryside
  • Prone to user disapproval
  • super creepy when you actually think about it
